YELLOW SCORING TABLE: (total of weighted yellow totals)
0 - 7 = Might Indicate You Currently Possess a Lower Skill Level in this Area
8 - 14 = Might Indicate You Currently Possess a Medium Skill Level in this Area
15 - 20 = Might Indicate You Currently Possess a High Skill Level in this Area
GREEN SCORING TABLE: (total of weighted green totals)
0 - 7 = Might Indicate You Have Very Little Interest in Improving This Skill Set
8 - 14 = Might Indicate You Will Use Your Experiences to Improve This Skill Set
15 - 20 = Might Indicate You Will Pursue Additional Training to Improve This Skill Set
